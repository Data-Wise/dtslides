Math & Equations

LaTeX Support

Mathematical equations are rendered beautifully using MathJax.

Inline math: \(E = mc^2\)

Display math:

\[ \frac{\partial \rho}{\partial t} + \nabla \cdot (\rho \mathbf{v}) = 0 \]

Aligned equations:

\[ \begin{align} f(x) &= a x^2 + b x + c \\ f'(x) &= 2 a x + b \end{align} \]

Theme Features

Custom Classes

The dtslides theme includes special utility classes:

  • Alert text for emphasis (.alert)
  • Custom Foreground (.fg)
  • Highlighted text (.highlight)
  • Note style (.note)
  • Warning style (.warning)

Standard Callouts

Note

Note callouts are great for non-critical information.

Important

Important callouts highlight critical warnings.

Tip

Tip callouts suggest best practices.

Theme-Specific Callouts

The theme provides custom semantic callouts:

Example: A worked problem or specific case study.

Question: A prompt for audience reflection or discussion.

Insight: A deeper observation or key takeaway.
